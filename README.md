# 📊 Power BI Dashboard – FinTech Performance Analysis

## 🔍 Overview
This project presents an interactive Power BI dashboard developed as part of my MBA project. The analysis focuses on evaluating the impact of Financial Technology (FinTech) on organizational performance using financial and statistical tools.

The dashboard provides insights into key financial indicators such as Assets Under Management (AUM), Non-Performing Assets (NPA), growth trends, and the relationship between FinTech investments and business outcomes.

---

## 🎯 Objectives
- To analyze financial performance using data visualization techniques  
- To evaluate the impact of FinTech adoption on key financial metrics  
- To identify trends, growth patterns, and correlations in financial data  
- To support decision-making using data-driven insights  

---

## 🛠 Tools & Technologies
- Power BI Desktop  
- Microsoft Excel  
- Data Analysis Techniques (Comparative, Trend, Growth Analysis)  
- Statistical Tools (Correlation & Regression Analysis)  

---

## 📈 Key Analysis Performed
- Comparative Financial Statement Analysis  
- Common Size Analysis  
- Trend Analysis  
- Growth Rate Analysis  
- Correlation Analysis  
- Regression Analysis  

---

## 💡 Key Insights
- FinTech investments show a positive relationship with AUM growth  
- Improved digital adoption contributes to operational efficiency  
- NPA trends indicate better risk management with technological integration  
- Strong correlation observed between FinTech spending and financial performance
